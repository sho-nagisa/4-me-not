# - The system should infer people, topics, tasks, dates, and interaction candidates later.
#
# Rules:
# - One "--- entry ---" block is one RawEntry.
# - You can add as many entries as needed in this single file.

# - Keep each entry realistic for a university student.
# - Include names, deadlines, classes, seminars, part-time jobs, job hunting,
#   circles, friends, relationships, and daily-life details where natural.
#
# Optional first line:
# date: YYYY-MM-DD
